# Code for [A Hybrid Cryptographic Simulator for Secure Communication Using QKD-E91 and AES-GCM]

This repository contains all code used in the Springer conference paper "[A Hybrid Cryptographic Simulator for Secure Communication Using QKD-E91 and AES-GCM]".

## Contents
- `notebooks/` : Google Colab notebook with all four graphs
- `images/` : PNG images of graphs used in the paper
- `tikz/` : LaTeX TikZ diagrams (if any)

## Instructions
Open the notebook in Google Colab to see and run the code. All graphs are already generated and saved as PNGs.
